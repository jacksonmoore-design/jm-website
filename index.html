WELCOME HOME

Hiya! Thank you for visiting my website.

I am a Chicago based artist and designer that loves patterns, textures and deep rich colors. I work in oil, fiber, installation, and now web design. I am interested in how I can create a "tactile", sensory experience through web design while still keeping the site easy to use.

As an artist and designer I am interested in the experience that a user can/will have with a website. How can I make it easier to navigate, how can I make it enjoyable, and ultimately how can I make a website a bit more fun? A place where a user will want to spend some time and explore.

I am very new to this medium so I created this website as a place for me to learn and understand all that is capable in the media of web design. It is a playground that will change and grow with time

I hope you enjoy it as much as I have had making it. Please let me know what you think on the Contact page.
